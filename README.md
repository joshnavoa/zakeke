# Zakeke Webflow Integration

Complete integration package for adding Zakeke product customization to Webflow projects.

## Files Overview

- **zakeke-config.js** - Main configuration and core integration logic
- **zakeke-order-api.js** - Order API integration for checkout
- **zakeke-webflow-integration.js** - Webflow-specific integration and UI setup
- **zakeke-styles.css** - Styling for the customizer modal and buttons
- **webflow-setup-instructions.md** - Detailed setup guide

## Quick Start

1. **Configure credentials** in `zakeke-config.js`
2. **Add files to Webflow** via Custom Code section
3. **Add data attributes** to product pages
4. **Test the integration**

See `webflow-setup-instructions.md` for detailed setup steps.

## Features

✅ Product customization via Zakeke iframe editor  
✅ Add customized products to cart  
✅ Edit products in cart  
✅ Checkout integration with Order API  
✅ Webflow-native cart support  
✅ Responsive design  

## API Integration

This package implements:
- **Customizer UI** - iframe-based product customizer
- **Cart API** - Add/update/remove cart items
- **Order API** - Create orders and process checkout
- **Product API** - Fetch product information
